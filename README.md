Open heart surgery simulator

A game from the point of view of a surgeon who must conduct open heart surgery on a patient. Gameplay involves using the keyboard to manipulate various medical tools to cut open the patientâ€™s heart, remove a tumor, and stitch it back together, similar to the popular board game Operation. The patient has three lives. If the surgeon makes a mistake, the patient loses a life, and if all lives are lost, the player loses.

To run the code on your device:
  1. Make sure you have the dependenciesnode.js installed correctly.
    a. Open a terminal in the project folder and run "npm install node".
    b. Install vite by running npm install --save-dev vite
  2. Launch the server with "npx vite"
  3. Navigate to http://localhost:5173/ to view
